# htmlsky

An HTML-only Bluesky frontend.

Just replace [bsky.app](https://bsky.app) with [htmlsky.app](https://htmlsky.app)!

Want JSON? `/raw/`
Want embeds? `/embed/`

## Self-hosting

Edit `fly.toml` to fit your needs.
